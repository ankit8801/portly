# Devendra Surve Portfolio

A portfolio platform and custom CMS built for showcasing graphic design projects through immersive case-study pages.

**Live Demo:** https://your-vercel-link.vercel.app

## About

This project was built to give graphic designer **Devendra Surve** complete control over publishing and presenting his work outside of platforms like Behance.

The website combines a public-facing portfolio with a custom-built content management system, allowing projects to be created, updated, and published without modifying the codebase.

Instead of relying on a fixed project structure, projects are composed from modular content blocks, making it possible to build detailed visual case studies tailored to each project.


## Features

### Portfolio Website

* Project gallery
* Dynamic project pages
* Responsive design
* SEO-friendly routing
* Smooth animations and transitions

### Admin CMS

* Secure authentication
* Project creation and editing
* Image uploads
* Block-based content builder
* Project publishing workflow

### Content Blocks

* Text
* Images
* Image Grids
* Video Embeds
